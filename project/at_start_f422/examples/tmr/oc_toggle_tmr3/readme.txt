  this demo is based on the at-start board, in this demo, pa6 pa7 pb0 pb1
  output waveform.
  c1dt update rate = tmr3 counter clock / c1dt_val = 610.35 hz
  c2dt update rate = tmr3 counter clock / c2dt_val = 1220.7 hz
  c3dt update rate = tmr3 counter clock / c3dt_val = 2441.4 hz
  c4dt update rate = tmr3 counter clock / c4dt_val = 4882.8 hz 

  for more detailed information. please refer to the application note document AN0085.
